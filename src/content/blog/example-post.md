# Welcome to Our Blog

This is an example blog post. You can create new blog posts by adding markdown files to the `src/content/blog/` directory.

## Features

- Easy to write and edit
- Version controlled
- Supports markdown formatting
- Frontmatter for metadata

## How to Use

1. Create a new `.md` file in `src/content/blog/`
2. Add frontmatter with title, date, and excerpt
3. Write your content in markdown
4. The blog system will automatically pick it up!
